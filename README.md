- Redux & Redux Toolkit & ReduxThunk
- @ant-design/react-native
- ESLint

## How to run this app

1. Put `.env` file on the root
2. Install dependencies

```
yarn
```

3. Run the app

```
yarn start
```

4. Scan the QR code on the screen by your phone :tada:
